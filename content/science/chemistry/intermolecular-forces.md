# 分子間作用力

## 凡得瓦力
### 偶極-偶極力（Dipole-Dipole Force）
- 極性分子之間的靜電吸引力
- 正端吸引負端，使分子定向排列
- 分子極性越大 → 偶極-偶極力越強
- 例：HCl、SO₂、CHCl₃ 之間的作用力
  - 💡 聯想：偶極-偶極力就像小磁鐵之間的吸引——正負相吸，排列整齊
### 倫敦分散力（London Dispersion Force）
- 存在於所有分子之間（包括非極性分子）
- 成因：電子雲的瞬間不均勻分布 → 瞬間偶極 → 誘導鄰近分子產生感應偶極
- 影響因素：
  - 分子量越大 → 電子數越多 → 分散力越強
  - 分子表面積越大 → 接觸面積越大 → 分散力越強
  - 直鏈分子 > 支鏈分子（表面積較大）
- 例：惰性氣體（He、Ne、Ar）的液化就靠分散力
  - 💡 聯想：分散力就像瞬間的心電感應——即使平時無極性，偶爾也會產生短暫的吸引
### 偶極-感應偶極力
- 極性分子靠近非極性分子時，使其電子雲偏移產生感應偶極
- 強度介於偶極-偶極力與分散力之間
- 例：HCl 溶於 CCl₄ 時的分子間作用力

## 氫鍵
### 形成條件
- 氫原子與**高電負度小原子**（F、O、N）形成共價鍵
- 該 H 原子再與另一個 F、O、N 原子上的孤對電子產生靜電吸引
- 表示為 X—H···Y（X、Y 為 F、O 或 N）
- 鍵能約 10～40 kJ/mol，比一般凡得瓦力強 5～10 倍
### 氫鍵的特性
- 具有**方向性**：接近直線型（X—H···Y 約 180°）效果最佳
- 具有**飽和性**：一個 H 原子通常只形成一個氫鍵
- 不是真正的化學鍵，而是特別強的分子間作用力
  - 💡 聯想：氫鍵就像朋友之間的握手——比路人擦肩（凡得瓦力）親密，但比夫妻（化學鍵）疏遠
### 氫鍵的重要實例
- **水（H₂O）**：
  - 每個水分子最多形成 4 個氫鍵（2 個 O—H 提供、2 個孤對電子接受）
  - 冰的結構：六角形開放結構 → 密度比液態水低（反常膨脹）
  - 4°C 時液態水密度最大
- **氟化氫（HF）**：形成鏈狀或環狀氫鍵聚合體
- **DNA 雙螺旋**：A-T 之間 2 個氫鍵，G-C 之間 3 個氫鍵
- **蛋白質**：氫鍵維持 α-螺旋與 β-摺板結構

## 分子間作用力的強度比較
### 強度排序
- 化學鍵（共價鍵、離子鍵、金屬鍵）>> 分子間作用力
- 氫鍵 > 偶極-偶極力 > 倫敦分散力（一般情形）
- 特例：大分子的倫敦分散力可能超過小分子的氫鍵
  - 例：I₂（分子量大，分散力強）的沸點高於 HF
### 典型鍵能範圍
- 共價鍵：150～1000 kJ/mol
- 氫鍵：10～40 kJ/mol
- 偶極-偶極力：5～25 kJ/mol
- 倫敦分散力：0.05～40 kJ/mol（隨分子量變化很大）
  - 💡 聯想：化學鍵像鐵鏈（超強），氫鍵像繩子（中等），凡得瓦力像便利貼（弱但多了也有用）
### 同類分子的比較
- 鹵化氫沸點：HF >> HI > HBr > HCl
  - HF 有氫鍵所以特別高
  - HI > HBr > HCl 因分子量遞增，分散力遞增
- 氫化物沸點異常：H₂O、HF、NH₃ 的沸點都異常偏高（氫鍵）

## 對物理性質的影響
### 沸點與熔點
- 分子間作用力越強 → 沸點、熔點越高
- 需要更多能量才能克服分子間吸引力
- 同系列：分子量增大 → 分散力增強 → 沸點升高
  - 例：烷類 CH₄ < C₂H₆ < C₃H₈ < ...
- 有氫鍵的物質沸點異常偏高
  - 例：H₂O（100°C）遠高於 H₂S（-60°C）
### 黏度
- 分子間作用力越強 → 黏度越大
- 能形成氫鍵的液體通常黏度較高
- 例：甘油（三個 —OH 基團，大量氫鍵）黏度很高
- 溫度升高 → 分子動能增大 → 黏度降低
  - 💡 聯想：黏度就像人群手拉手——分子間拉力越大（作用力越強），流動越困難
### 表面張力
- 表面分子只受向內的淨引力 → 表面積趨於最小
- 分子間作用力越強 → 表面張力越大
- 水的表面張力特別大（氫鍵的貢獻）
  - 水黽能站在水面上、硬幣能浮在水面上
- 加入界面活性劑可降低表面張力
### 毛細現象
- 附著力 > 內聚力 → 液面上升（如水在玻璃管中）→ 凹液面
- 內聚力 > 附著力 → 液面下降（如水銀在玻璃管中）→ 凸液面
- 管徑越細 → 上升（下降）越高

## 分子間作用力與化學鍵的區別
### 本質差異
- **化學鍵**：原子之間的強烈交互作用，形成分子或晶體
  - 涉及電子的轉移（離子鍵）或共用（共價鍵）
  - 鍵能：100～1000 kJ/mol
- **分子間作用力**：分子與分子之間的弱交互作用
  - 涉及電荷分布的靜電吸引
  - 鍵能：0.05～40 kJ/mol
### 對性質的不同影響
- 化學變化：斷裂化學鍵（需要大量能量）
- 物理變化（熔化、沸騰、溶解）：克服分子間作用力
- 例：水沸騰時 O—H 共價鍵不斷裂，只是分子間的氫鍵被破壞
  - 💡 聯想：煮水時水分子「鬆手放開朋友」（氫鍵斷裂），但每個水分子本身完好無缺（共價鍵不變）
### 判斷依據
- 分子間作用力決定物質的**物理性質**（沸點、溶解度等）
- 化學鍵決定物質的**化學性質**（反應活性、穩定性等）
- 離子晶體、金屬晶體、網狀固體中不存在獨立分子 → 無分子間作用力的概念
  - 這些晶體的熔沸點由化學鍵直接決定
