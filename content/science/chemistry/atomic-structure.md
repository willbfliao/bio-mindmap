# 原子構造

## 氫原子光譜
### 光譜的產生
- 氫原子受激發後，電子從高能階躍遷到低能階時放出特定波長的光
- 每條譜線對應特定的能階躍遷
- 氫原子光譜為**線光譜**（不連續），而非連續光譜
  - 💡 聯想：氫原子光譜就像鋼琴的琴鍵——只有特定音符（波長）會被彈出
### 光譜系列
- **賴曼系列（Lyman）**：電子躍遷至 n=1（基態）
  - 放出紫外光，波長最短、能量最大
- **巴耳末系列（Balmer）**：電子躍遷至 n=2
  - 部分落在可見光區，為最早被發現的系列
  - 可見光譜線：H_α（紅）、H_β（藍綠）、H_γ（藍紫）、H_δ（紫）
- **乃辛系列（Paschen）**：電子躍遷至 n=3，落在紅外光區
- **布拉乃系列（Brackett）**：電子躍遷至 n=4，落在紅外光區
- **乃輔系列（Pfund）**：電子躍遷至 n=5，落在紅外光區
### 里德柏公式
- (1/λ) = R_H ((1/n₁²) - (1/n₂²))
- R_H = 1.097 × 10⁷  m⁻¹（里德柏常數）
- n₁ < n₂，n₁ 為終態，n₂ 為初態
  - 💡 聯想：這個公式就像一把尺——能精確量出氫原子每條光譜線的波長

## 波耳模型
### 基本假設
- 電子在固定的**量子化軌道**上運動，不會輻射能量
- 電子角動量量子化：L = n(h/2π)，n = 1, 2, 3, …
- 電子在不同軌道間躍遷時才會吸收或放出能量
### 能階公式
- 氫原子第 n 能階的能量：Eₙ = -(13.6/n²) eV
- n=1（基態）：E₁ = -13.6 eV，能量最低、最穩定
- n=2：E₂ = -3.4 eV
- n=∞：E_∞ = 0 eV，電子完全脫離原子（游離）
### 軌道半徑
- 第 n 軌道半徑：rₙ = n² × a₀
- a₀ = 0.529 Å（波耳半徑）
- 軌道間距隨 n 增大而增大
### 能階躍遷
- 吸收光子（激發）：E(光子) = E(高) - E(低) = hν
- 放出光子（退激）：電子從高能階掉回低能階
- 游離能：使基態電子脫離原子所需的最小能量 = 13.6 eV
  - 💡 聯想：波耳模型像行星繞太陽——電子只能在特定軌道運行，要跳到別的軌道就得吸收或釋放能量
### 波耳模型的限制
- 僅適用於氫原子及類氫離子（He⁺、Li²⁺）
- 無法解釋多電子原子的光譜
- 無法解釋精細結構與磁場中的譜線分裂

## 量子力學基礎
### 波粒二象性
- 德布羅意假說：所有物質都有波動性
- 物質波波長：λ = (h/mv)
- 電子具有波動性 → 電子繞核的駐波條件導出量子化
### 測不準原理
- 海森堡測不準原理：Δ x · Δ p ≥ (h/4π)
- 不可能同時精確測定粒子的位置與動量
- 因此用「電子雲」（機率密度）取代確定軌道
  - 💡 聯想：就像用手電筒照飛蛾——你看到的只是飛蛾「可能出現的區域」，而非精確位置
### 四個量子數
- **主量子數 n**：n = 1, 2, 3, …
  - 決定電子的能量與軌域大小
  - n 越大，離核越遠、能量越高
- **角量子數 l**：l = 0, 1, 2, …, (n-1)
  - 決定軌域形狀
  - l = 0（s）、l = 1（p）、l = 2（d）、l = 3（f）
- **磁量子數 mₗ**：mₗ = -l, …, 0, …, +l
  - 決定軌域在空間中的方向
  - 每個 l 值有 (2l+1) 個方向
- **自旋量子數 mₛ**：mₛ = +(1/2) 或 -(1/2)
  - 描述電子的自旋方向（上旋或下旋）
  - 同一軌域最多容納 2 個自旋相反的電子（乃乃不相容原理）

## 原子軌域
### s 軌域
- 形狀：球形對稱
- 每個能階有 1 個 s 軌域
- 1s：最小球形；2s：較大球形，中間有一個節面
- 節面數 = n - 1（主節面數）
### p 軌域
- 形狀：啞鈴形（雙葉形）
- 每個能階（n ≥ 2）有 3 個 p 軌域：pₓ、p_y、p_z
- 三個 p 軌域互相垂直，能量相同（簡併）
- 節面：通過原子核的一個平面
### d 軌域
- 形狀：花瓣形（四葉草形或甜甜圈形）
- 每個能階（n ≥ 3）有 5 個 d 軌域
- dxy、dxz、dyz：花瓣在座標軸之間
- dx²-y²：花瓣沿 x、y 軸
- dz²：沿 z 軸的啞鈴加環形帶
  - 💡 聯想：s 軌域像氣球、p 軌域像啞鈴、d 軌域像四葉草——形狀越來越複雜
### 各殼層軌域數
- 第 n 殼層：共有 n² 個軌域，最多容納 2n² 個電子
- n=1：1 個軌域（1s），最多 2 個電子
- n=2：4 個軌域（2s + 三個 2p），最多 8 個電子
- n=3：9 個軌域（3s + 三個 3p + 五個 3d），最多 18 個電子

## 電子組態
### 構築原理（Aufbau Principle）
- 電子由低能階往高能階依序填入
- 填入順序：1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → …
- 能階高低判斷：(n + l) 值小者優先；(n + l) 相同時，n 小者優先
  - 💡 聯想：電子就像排隊搶位子——先搶能量最低的好位子
### 乎乃定則（Hund's Rule）
- 電子填入相同能量的軌域時，先各佔一個軌域且自旋平行
- 半滿與全滿具有額外穩定性
- 例：碳 1s² 2s² 2p²，兩個 2p 電子分佔不同軌域
### 乃乃不相容原理（Pauli Exclusion Principle）
- 同一原子中沒有兩個電子具有完全相同的四個量子數
- 同一軌域最多容納 2 個電子，自旋必須相反（↑↓）
### 常見元素的電子組態
- H：1s¹
- C：1s² 2s² 2p²
- N：1s² 2s² 2p³（半滿，穩定）
- Ne：1s² 2s² 2p⁶（全滿，穩定）
- Fe：[Ar] 3d⁶ 4s²
### 例外電子組態
- Cr：[Ar] 3d⁵ 4s¹（非 3d⁴ 4s²）
  - 3d 半滿（3d⁵）具有額外穩定性
- Cu：[Ar] 3d¹⁰ 4s¹（非 3d⁹ 4s²）
  - 3d 全滿（3d¹⁰）具有額外穩定性
  - 💡 聯想：Cr 和 Cu 就像教室裡的特殊座位——半滿整排或全滿整排特別穩定，電子寧可調位子

## 週期表與電子組態的關係
### 區塊分類
- **s 區**：ⅠA、ⅡA 族，最外層填 s 軌域
- **p 區**：ⅢA～ⅧA 族（含惰性氣體），最外層填 p 軌域
- **d 區**：過渡金屬（ⅢB～ⅡB），填入 (n-1)d 軌域
- **f 區**：鑭系與錒系元素，填入 (n-2)f 軌域
### 價電子與族的關係
- 主族元素：族數 = 價電子數
  - ⅠA 族有 1 個價電子，ⅥA 族有 6 個價電子
- 過渡金屬：價電子包含 ns 和 (n-1)d 電子
### 週期性質與電子組態
- 同族元素電子組態相似 → 化學性質相似
- 同週期從左到右，有效核電荷增大 → 原子半徑縮小
- 離子的電子組態：陽離子先失去最外層 s 電子，再失去 d 電子
  - 例：Fe²⁺：[Ar] 3d⁶（先失去 4s²）
  - 💡 聯想：週期表就像一棟電子公寓——每一層（週期）住的人越來越多，每一行（族）的住戶有相似的個性
