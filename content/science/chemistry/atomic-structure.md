# 原子構造

## 氫原子光譜
### 光譜的產生
- 電子由高能階躍遷至低能階 → 放出特定波長的光
- 每條譜線對應一組能階躍遷
- **線光譜**（不連續），非連續光譜
  - 💡 聯想：氫原子光譜就像鋼琴的琴鍵——只有特定音符（波長）會被彈出
### 光譜系列
- **賴曼系列（Lyman）**：電子躍遷至 $n=1$（基態）
  - 放出紫外光，波長最短、能量最大
- **巴耳末系列（Balmer）**：電子躍遷至 $n=2$
  - 部分落在可見光區，為最早被發現的系列
  - 可見光譜線：$\text{H}_{\alpha}$（紅）、$\text{H}_{\beta}$（藍綠）、$\text{H}_{\gamma}$（藍紫）、$\text{H}_{\delta}$（紫）
- **帕申系列（Paschen）**：電子躍遷至 $n=3$，落在紅外光區
- **布拉克特系列（Brackett）**：電子躍遷至 $n=4$，落在紅外光區
- **普芬系列（Pfund）**：電子躍遷至 $n=5$，落在紅外光區
  - 💡 聯想：光譜系列像住在不同樓層回家——回1樓（賴曼）最辛苦能量最大，回2樓（巴耳末）剛好看得見
### 里德柏公式
- $\frac{1}{\lambda} = R_{\text{H}} \left( \frac{1}{n_1^{2}} - \frac{1}{n_2^{2}} \right)$
- $R_{\text{H}} = 1.097 \times 10^{7} \text{ m}^{-1}$（里德柏常數）
- $n_1 < n_2$，$n_1$ 為終態，$n_2$ 為初態
  - 💡 聯想：這個公式就像一把尺——能精確量出氫原子每條光譜線的波長

## 波耳模型
### 基本假設
- 電子在量子化軌道上運動，不輻射能量
  - 波耳假設電子在特定軌道上繞原子核作圓周運動（類似行星模型）
- 角動量量子化：$L = \frac{nh}{2\pi}$，$n = 1, 2, 3, \ldots$
- 躍遷時才吸收或放出能量（$\Delta E = h\nu$）
### 能階公式
- 氫原子第 $n$ 能階的能量：$E_n = -\frac{13.6}{n^{2}}$ eV
- $n=1$（基態）：$E_1 = -13.6$ eV，能量最低、最穩定
- $n=2$：$E_2 = -3.4$ eV
- $n=\infty$：$E_{\infty} = 0$ eV，電子完全脫離原子（游離）
### 軌道半徑
- 第 $n$ 軌道半徑：$r_n = n^{2} \times a_0$
- $a_0 = 0.529$ Å（波耳半徑）
- 軌道間距隨 $n$ 增大而增大
### 能階躍遷
- 吸收光子（激發）：$E_{\text{光子}} = E_{\text{高}} - E_{\text{低}} = h\nu$
- 放出光子（退激）：電子從高能階掉回低能階
- 游離能：使基態電子脫離原子所需的最小能量 $= 13.6$ eV
  - 💡 聯想：波耳模型像行星繞太陽——電子只能在特定軌道運行，要跳到別的軌道就得吸收或釋放能量
### 波耳模型的限制
- 僅適用於氫原子及類氫離子（$\text{He}^{+}$、$\text{Li}^{2+}$）
- 無法解釋多電子原子的光譜
- 無法解釋精細結構與磁場中的譜線分裂
- 認為電子在特定軌道上作圓周運動，但根據量子理論，電子並非以此方式存在

## 量子力學基礎
### 波粒二象性
- 德布羅意假說：物質皆具波動性
- 物質波波長：$\lambda = \frac{h}{mv}$
- 電子繞核的駐波條件 → 量子化的來源
  - 💡 聯想：電子波就像吉他弦振動——只有「剛好整數個波長」的振動才穩定存在
### 測不準原理
- 海森堡：$\Delta x \cdot \Delta p \geq \frac{h}{4\pi}$
- 位置與動量無法同時精確測定
- 用「電子雲」（機率密度）取代確定軌道
  - 電子雲密度越大 → 電子出現機率越高；密度越小 → 機率越低
- **軌域（orbital）**：由原子核往外延伸至電子出現機率約 90% 的空間範圍
  - 💡 聯想：就像用手電筒照飛蛾——你看到的只是飛蛾「可能出現的區域」，而非精確位置
### 四個量子數
- **主量子數 $n$**：$n = 1, 2, 3, \ldots$
  - 決定電子的能量與軌域大小（殼層 shell）
  - $n$ 越大，離核越遠、能量越高、軌域範圍越大
  - 殼層名稱對照：$n=1$（K）、$n=2$（L）、$n=3$（M）、$n=4$（N）、$n=5$（O）
- **角量子數 $l$**：$l = 0, 1, 2, \ldots, (n-1)$
  - 決定軌域形狀（又稱副殼層 subshell）
  - $l = 0$（s，球形）、$l = 1$（p，啞鈴形）、$l = 2$（d，花瓣形）、$l = 3$（f，複雜形）
  - 每個 $n$ 值的殼層含有 $n$ 種軌域
  - 軌域類型一覽：
    - $l=0$（s）：球形，$n \geq 1$ 起始，每組 1 個
    - $l=1$（p）：啞鈴形，$n \geq 2$ 起始，每組 3 個
    - $l=2$（d）：花瓣形，$n \geq 3$ 起始，每組 5 個
    - $l=3$（f）：複雜形，$n \geq 4$ 起始，每組 7 個
- **磁量子數 $m_l$**：$m_l = -l, \ldots, 0, \ldots, +l$
  - 決定軌域在空間中的方向與個數
  - 每個 $l$ 值有 $(2l+1)$ 個軌域
  - $l=0$：$m_l = 0$ → s 軌域只有 1 個
  - $l=1$：$m_l = +1, 0, -1$ → p 軌域有 3 個
  - $l=2$：$m_l = +2, +1, 0, -1, -2$ → d 軌域有 5 個
- **自旋量子數 $m_s$**：$m_s = +\frac{1}{2}$（上旋）或 $-\frac{1}{2}$（下旋）
  - 描述電子的自旋方向
  - 同一軌域最多容納 2 個自旋相反的電子（包立不相容原理）
  - 💡 聯想：四個量子數就像飯店地址——$n$ 是樓層、$l$ 是房型（套房/雅房）、$m_l$ 是房號、$m_s$ 是床位（上鋪/下鋪）

## 原子軌域
### s 軌域
- 形狀：球形對稱
- 每一個 $n$ 值皆有 1 個 s 軌域
- $n$ 愈大 → $ns$ 軌域電子雲分布半徑愈大、能量（能階）愈高
  - 例：能階 $1s < 2s < 3s$
- 相同半徑的球面上，電子出現機率相等，與方向無關 → **無方向性**
- $1s$：最小球形；$2s$：較大球形，中間有一個節面
- 節面數 $= n - 1$
### p 軌域
- 形狀：啞鈴形（雙葉形）
- $n \geq 2$ 時，每一個 $n$ 值皆有 3 個 p 軌域：$p_x$、$p_y$、$p_z$
- 分別位在 $x$、$y$、$z$ 三個互相垂直的軸上 → **有方向性**
- 同一 $n$ 值之三個 p 軌域能量相同（簡併）
  - 例：$2p_x = 2p_y = 2p_z$
- $n$ 愈大 → 電子雲分布離原子核愈遠、能量（能階）愈高
  - 例：能階 $2p < 3p < 4p$
- **節面（node）**：電子雲出現機率為 0 的區域
  - p 軌域恆有 1 個含原子核的角節面（決定方向性）
  - 總節面數 $= n - 1$（角節面 1 個 + 徑向節面 $n - 2$ 個）
  - 例：$2p$ 共 1 個節面（皆為角節面）；$3p$ 共 2 個節面（1 角 + 1 徑向）
### d 軌域
- 形狀：花瓣形（四葉草形或甜甜圈形）
- $n \geq 3$ 時，每一個 $n$ 值皆有 5 個 d 軌域，有方向性
- $d_{xy}$、$d_{xz}$、$d_{yz}$：花瓣在座標軸之間
- $d_{x^{2}-y^{2}}$：花瓣沿 x、y 軸
- $d_{z^{2}}$：沿 z 軸的啞鈴加環形帶
  - 💡 聯想：s 軌域像氣球、p 軌域像啞鈴、d 軌域像四葉草——形狀越來越複雜
### 殼層軌域總整理
- 第 $n$ 殼層：有 $n$ 種軌域類型、共 $n^{2}$ 個軌域、最多容納 $2n^{2}$ 個電子
- $n=1$（K 殼層）：$l=0$（1s），$m_l = 0$，共 1 個軌域，最多 2 個電子
- $n=2$（L 殼層）：$l=0,1$（2s, 2p），共 4 個軌域，最多 8 個電子
- $n=3$（M 殼層）：$l=0,1,2$（3s, 3p, 3d），共 9 個軌域，最多 18 個電子
- $n=4$（N 殼層）：$l=0,1,2,3$（4s, 4p, 4d, 4f），共 16 個軌域，最多 32 個電子
### 軌域能量比較
#### 單電子原子（如 H、$\text{He}^+$、$\text{Li}^{2+}$）
- 能量只取決於主量子數 $n$
- 相同 $n$ 的所有軌域能量相同（簡併）
- 能量順序：$1s < 2s = 2p < 3s = 3p = 3d < 4s = 4p = 4d = 4f$
#### 多電子原子（如 He、Li、Be、B……）
- 因屏蔽效應與穿透效應，能量取決於 $(n+l)$ 值
- $(n+l)$ 值越大 → 能量越高
- $(n+l)$ 相同時 → $n$ 越大者能量越高
- 能量順序：$1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s < 4f < 5d < 6p$
  - 💡 聯想：$(n+l)$ 規則就像算「能量分數」——分數相同時，住在低樓層（小 $n$）的能量更低

## 電子組態
### 電子組態表示法
- 原子的電子占有軌域的排列方式，以 $nl^{x}$ 表示
  - $n$：主量子數
  - $l$：軌域類型（s / p / d / f）
  - $x$：該軌域中的電子數量
- 例：$1s^{2}$ 表示 $n=1$、$l=0$（s 軌域）、有 2 個電子
- 各軌域最多電子數：s 至多 2 個、p 至多 6 個、d 至多 10 個、f 至多 14 個
### 屏蔽效應與穿透效應
- **屏蔽效應**：內層電子減弱外層電子感受的核電荷 → 有效核電荷 $Z_{\text{eff}} = Z - \sigma$
  - 內層電子屏蔽效果大；同層電子屏蔽效果小
- **穿透效應**：s 軌域電子更接近原子核 → 受屏蔽較少 → 能量較低
  - 同一殼層能量：$s < p < d < f$
  - 解釋為什麼 $4s$ 比 $3d$ 先填入
  - 💡 聯想：s 電子像喜歡往前擠的學生，能擠到老師（原子核）旁邊，所以被照顧（能量低）
### 電子填入軌域三大原則
#### 構築原理（Aufbau Principle）
- 電子由低能階往高能階依序填入，低能階軌域填滿後再填高能階
- 填入順序：$1s \to 2s \to 2p \to 3s \to 3p \to 4s \to 3d \to 4p \to 5s \to 4d \to \ldots$
- 能階高低判斷：$(n + l)$ 值小者優先；$(n + l)$ 相同時，$n$ 小者優先
- 違反構築原理的電子組態為**激發態**
  - 例：$_5\text{B}$：$1s^2 2s^2 2p^1$（基態）；$1s^2 2s^2 3s^1$（激發態）
  - 💡 聯想：電子就像排隊搶位子——先搶能量最低的好位子
#### 包立不相容原理（Pauli Exclusion Principle）
- 同一原子中沒有兩個電子具有完全相同的四個量子數
- 同一軌域最多容納 2 個電子，自旋必須相反（↑↓）
- 違反包立不相容原理的電子組態**不被允許存在**（非激發態，而是根本不可能）
  - 例：$1s^3$ → 不存在（一個軌域不可能塞 3 個電子）
- 影響：決定各軌域最多可容納的電子數
#### 洪德定則（Hund's Rule）
- 電子填入能階相同的副殼層軌域時，先以相同的自旋方向分別填入不同方向的軌域（先不成對）
- 等到各軌域均有 1 個電子後，才允許自旋方向相反之電子填入而成對
- 半滿與全滿具有額外穩定性
- 違反洪德定則的電子組態為**激發態**
  - 例：$_6\text{C}$：$1s^2 2s^2 2p^2$，兩個 $2p$ 電子應分佔 $2p_x$（↑）和 $2p_y$（↑），而非擠在同一個 $2p_x$（↑↓）
### 基態、激發態與不存在的組態
- **基態**：符合三大原則，能量最低、最穩定的排列
- **激發態**：違反構築原理或洪德定則，電子可存在但不在最低能量排列
- **不存在的組態**：違反包立不相容原理，根本不可能存在
  - 💡 聯想：激發態像是學生亂坐位子——不是最好的安排但可以存在；違反包立則像一個座位硬擠三人——根本坐不下
### 電子組態書寫規則
- **書寫順序**（與填入順序不同！）：
  - 同一主量子數 $n$ 內，依 s → p → d → f 排列
  - 再依 $n$ 由小到大排列
  - 例：Fe 填入順序為 $\ldots 4s^2\ 3d^6$，書寫為 $[\text{Ar}]\ 3d^6\ 4s^2$
### 鈍氣核心簡式
- 以前一週期的鈍氣符號加中括號，代替全填滿的內殼層電子組態（原子序 $\geq 3$ 均適用）
- $[\text{He}] = 1s^2$（2 個電子）
- $[\text{Ne}] = 1s^2\ 2s^2\ 2p^6$（10 個電子）
- $[\text{Ar}] = 1s^2\ 2s^2\ 2p^6\ 3s^2\ 3p^6$（18 個電子）
- $[\text{Kr}] = [\text{Ar}]\ 3d^{10}\ 4s^2\ 4p^6$（36 個電子）
- **範例**：
  - Si（$Z=14$）：$1s^2 2s^2 2p^6 3s^2 3p^2$ → 簡化 $[\text{Ne}]\ 3s^2\ 3p^2$
  - Ca（$Z=20$）：簡化 $[\text{Ar}]\ 4s^2$
  - Fe（$Z=26$）：填入 $\ldots 4s^2\ 3d^6$ → 書寫 $[\text{Ar}]\ 3d^6\ 4s^2$
  - As（$Z=33$）：填入 $\ldots 4s^2\ 3d^{10}\ 4p^3$ → 書寫 $[\text{Ar}]\ 3d^{10}\ 4s^2\ 4p^3$

### 例外電子組態
- Cr：$[\text{Ar}]\ 3d^{5}\ 4s^{1}$（非 $3d^{4}\ 4s^{2}$）→ $3d$ 半滿具有額外穩定性
- Cu：$[\text{Ar}]\ 3d^{10}\ 4s^{1}$（非 $3d^{9}\ 4s^{2}$）→ $3d$ 全滿具有額外穩定性
  - 💡 聯想：Cr 和 Cu 就像教室裡的特殊座位——半滿整排或全滿整排特別穩定，電子寧可調位子
### 離子的電子組態
#### 陽離子（失去電子）
- 先寫出原子的基態電子組態
- 由**最外層**（即主量子數最大的殼層）中先移去電子，而非最高能階軌域
- 範例：
  - $_{21}\text{Sc} \to \text{Sc}^{3+} + 3e^-$：$[\text{Ar}]3d^1 4s^2 \to [\text{Ar}]$（先移 $4s$ 的 2 個，再移 $3d$ 的 1 個）
  - $_{26}\text{Fe} \to \text{Fe}^{2+} + 2e^-$：$[\text{Ar}]3d^6 4s^2 \to [\text{Ar}]3d^6$（移去 $4s$ 的 2 個）
#### 陰離子（獲得電子）
- 先寫出原子的基態電子組態
- 將電子加入最外層的軌域
- 範例：
  - $_{17}\text{Cl} + e^- \to \text{Cl}^-$：$[\text{Ne}]3s^2 3p^5 \to [\text{Ne}]3s^2 3p^6 = [\text{Ar}]$
  - $_{16}\text{S} + 2e^- \to \text{S}^{2-}$：$[\text{Ne}]3s^2 3p^4 \to [\text{Ne}]3s^2 3p^6 = [\text{Ar}]$
#### 等電子物種
- **主族（A 族）**：相同電子數 → 電子組態**必相同**
  - 例：$\text{S}^{2-}$、$\text{Cl}^-$、Ar、$\text{K}^+$、$\text{Ca}^{2+}$ 皆為 18 個電子，組態均為 $[\text{Ne}]3s^2 3p^6$
- **過渡金屬（B 族）**：相同電子數 → 電子組態**不一定相同**
  - 原因：中性原子的 $4s$ 能量低於 $3d$，但形成陽離子時 $3d$ 反而穩定，因此「先脫 $4s$ 再脫 $3d$」可能讓組態與同電子數的中性原子或其他離子不同
  - 例：$_{23}\text{V}$（$[\text{Ar}]3d^3 4s^2$）與 $_{25}\text{Mn}^{2+}$（$[\text{Ar}]3d^5$）電子數均為 23，組態不同
  - 例：$_{26}\text{Fe}^{2+}$（$[\text{Ar}]3d^6$）與 $_{27}\text{Co}^{3+}$（$[\text{Ar}]3d^6$）電子數均為 24，組態相同
  - 💡 聯想：A 族離子像制服——穿一樣就長一樣；B 族離子像便服——人數相同穿搭不一定相同

## 週期表與電子組態的關係
### 區塊分類
- **s 區**：ⅠA、ⅡA 族，最外層填 s 軌域
- **p 區**：ⅢA～ⅧA 族（含惰性氣體），最外層填 p 軌域
- **d 區**：過渡金屬（ⅢB～ⅡB），填入 $(n-1)d$ 軌域
- **f 區（內過渡元素）**：
  - 鑭系元素（$Z = 57 \sim 71$）：屬於第六週期，填入 $4f$ 軌域
  - 錒系元素（$Z = 89 \sim 103$）：屬於第七週期，填入 $5f$ 軌域
  - 為縮短週期表長度，將其置於週期表底部
### 各週期元素數
- 第 1 週期：2 個元素（$1s$）→ 鈍氣 He（$Z=2$）
- 第 2 週期：8 個元素（$2s + 2p$）→ 鈍氣 Ne（$Z=10$）
- 第 3 週期：8 個元素（$3s + 3p$）→ 鈍氣 Ar（$Z=18$）
- 第 4 週期：18 個元素（$4s + 3d + 4p$）→ 鈍氣 Kr（$Z=36$）
- 第 5 週期：18 個元素（$5s + 4d + 5p$）→ 鈍氣 Xe（$Z=54$）
- 第 6 週期：32 個元素（$6s + 4f + 5d + 6p$）→ 鈍氣 Rn（$Z=86$）
- 第 7 週期：32 個元素（$7s + 5f + 6d + 7p$）→ Og（$Z=118$）
### 價電子數與族的對應
#### A 族（主族元素）
- 又稱典型元素，為 s-區與 p-區元素
- 價電子數 $=$ 所屬 A 族族數（He 除外）
- ⅠA：$ns^1$（1 個價電子）
- ⅡA：$ns^2$（2 個價電子）
- ⅢA：$ns^2 np^1$（3 個價電子）
- ⅣA：$ns^2 np^2$（4 個價電子）
- ⅤA：$ns^2 np^3$（5 個價電子）
- ⅥA：$ns^2 np^4$（6 個價電子）
- ⅦA：$ns^2 np^5$（7 個價電子）
- ⅧA：$ns^2 np^6$（8 個價電子，He 為 $1s^2$）
#### B 族（過渡元素）
- 又稱 d-區元素，全為金屬元素
- 價電子數 $=$ $ns$ 與 $(n-1)d$ 電子數和
- 第四週期過渡元素（Sc～Zn）：
  - ⅢB（Sc）：$3d^1 4s^2$，3 個價電子
  - ⅣB（Ti）：$3d^2 4s^2$，4 個價電子
  - ⅤB（V）：$3d^3 4s^2$，5 個價電子
  - ⅥB（Cr）：$3d^5 4s^1$，6 個價電子
  - ⅦB（Mn）：$3d^5 4s^2$，7 個價電子
  - ⅧB（Fe, Co, Ni）：$3d^{6 \sim 8} 4s^2$，8～10 個價電子
  - ⅠB（Cu）：$3d^{10} 4s^1$，11 個價電子
  - ⅡB（Zn）：$3d^{10} 4s^2$，12 個價電子
- 第五～七週期的過渡元素不完全依循第四週期規則，高中不詳加討論
### 電子組態速寫法
- 利用元素在週期表的位置快速寫出電子組態
- **步驟一**：前一週期的鈍氣為核心，寫在前方
- **步驟二**：從族數判斷價電子組態，寫在後方
- **步驟三**：補滿中間的 d 或 f 軌域電子
  - 若鈍氣與元素原子序差 10 個 → 補 $(n-1)d^{10}$
  - 若差 24 個 → 補 $(n-2)f^{14}(n-1)d^{10}$
  - $n$ 為最外層主量子數（即所在週期）
- **範例**：
  - Se（$Z=34$，第 4 週期 ⅥA）：
    - ❶ $[\text{Ar}]$（$Z=18$）
    - ❷ ⅥA → $4s^2 4p^4$
    - ❸ $34 - 18 - 6 = 10$ → 補 $3d^{10}$
    - 結果：$[\text{Ar}]\ 3d^{10}\ 4s^2\ 4p^4$
  - Pb（$Z=82$，第 6 週期 ⅣA）：
    - ❶ $[\text{Xe}]$（$Z=54$）
    - ❷ ⅣA → $6s^2 6p^2$
    - ❸ $82 - 54 - 4 = 24$ → 補 $4f^{14} 5d^{10}$
    - 結果：$[\text{Xe}]\ 4f^{14}\ 5d^{10}\ 6s^2\ 6p^2$
  - 💡 聯想：速寫法就像寄信地址——先寫國家（鈍氣核心）、再寫城市（外層）、最後補街道（中間 d/f 軌域）
### 週期性質與電子組態
- 同族元素電子組態相似 → 化學性質相似
- 同週期從左到右，有效核電荷增大 → 原子半徑縮小
  - 💡 聯想：週期表就像一棟電子公寓——每一層（週期）住的人越來越多，每一行（族）的住戶有相似的個性
### 逐步游離能
- 將原子中電子一個一個移除，每次所需能量遞增
- $IE_1 < IE_2 < IE_3 < \ldots$：電子越少，有效核電荷越大，越難拔
- **能量跳躍**：跨殼層時游離能突然大幅上升
  - 例：Na 的 $IE_1$ 小，$IE_2$ 突然大幅跳升 → Na 有 1 個價電子
  - 例：Mg 的 $IE_1$、$IE_2$ 相近，$IE_3$ 突然跳升 → Mg 有 2 個價電子
- 學測常考：從游離能跳躍推斷價電子數與族別
  - 💡 聯想：就像剥洋蔥——前幾層很好剥（價電子），突然遇到很硬的一層（內層電子）就剥不動了
